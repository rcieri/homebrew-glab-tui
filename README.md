# Homebrew tap for glab-tui

## Usage

```sh
brew tap rcieri/glab-tui
brew install glab-tui
```

## Updating

The formula is auto-updated daily by a GitHub Actions workflow that checks for
new releases of [glab-tui](https://github.com/rcieri/glab-tui) and updates the
URLs and checksums automatically. You can also trigger it manually from the
Actions tab.

To check for a newer version manually:

```sh
brew livecheck glab-tui
```
